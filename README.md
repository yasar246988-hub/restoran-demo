# 🎯 Restoran Yönetim Sistemi - DEMO v2.0.0

## ✨ Yeni Özellikler

### QR Menü Sistemi
- ✅ **7 Farklı Tema:** Modern, Warm, Fresh, Elegant, Vibrant, Vintage, Minimal
- ✅ **4 Farklı Layout:** Grid, List, Masonry, Horizontal
- ✅ **3 Dil Desteği:** Türkçe, İngilizce, Arapça
- ✅ **Logo Yükleme:** Kafe logosu ekleme
- ✅ **Tema Özelleştirme:** Kafe adı, slogan, renkler

### Güvenlik
- ⚠️ **Demo Modu:** Konum kontrolü DEVRE DIŞI (test için)
- ✅ **Session Yönetimi:** 8 saat, 10 sipariş hakkı
- ✅ **Token Tabanlı:** Güvenli sipariş sistemi

## 🚀 Railway Deployment
